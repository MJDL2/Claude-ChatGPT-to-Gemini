📥 입력 파일 위치

이 폴더에 conversations.json 파일을 복사하세요.

📋 지원 파일:
✅ Claude conversations.json
✅ ChatGPT conversations.json

❌ 지원하지 않는 파일:
- 기타 AI 서비스 파일
- 텍스트 파일
- 다른 이름의 JSON 파일

💡 파일 다운로드 방법:

🤖 Claude:
1. claude.ai → Settings → Data export
2. "Download conversations" 클릭
3. conversations.json 다운로드

💬 ChatGPT:  
1. chat.openai.com → Settings → Data controls
2. "Export data" 클릭
3. 이메일로 받은 ZIP에서 conversations.json 추출

⚠️ 주의사항:
- 파일명이 정확히 "conversations.json"이어야 함
- 파일 크기가 0KB가 아니어야 함
- 올바른 JSON 형식이어야 함

📁 예시 경로:
C:\...\Claude-ChatGPT-to-Gemini\input\conversations.json
