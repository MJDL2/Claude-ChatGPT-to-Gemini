📤 결과 파일 생성 위치

process_conversations.bat 실행 후 이 폴더에 다음 파일들이 생성됩니다:

📊 기본 결과 파일:
- conversations_organized.txt     (정리된 전체 대화)
- conversations_index.txt         (대화 목록 인덱스)  
- processing_report.txt           (처리 결과 요약)

📄 NotebookLM용 분할 파일:
- notebooklm_part_01_Programming.txt
- notebooklm_part_02_Data_Analysis.txt  
- notebooklm_part_03_AI_Tech.txt
- notebooklm_part_04_Problem_Solving.txt
- notebooklm_part_05_General.txt
- notebooklm_part_06_Daily_Mixed.txt
- notebooklm_index.txt            (분할 파일 가이드)

🎯 사용법:
1. NotebookLM (https://notebooklm.google.com) 접속
2. 새 노트북 생성
3. 관심 있는 분할 파일들을 업로드
4. AI에게 대화 내용에 대해 질문

💡 추천 질문:
- "내가 프로그래밍에 대해 어떤 고민을 했나요?"
- "데이터 분석 프로젝트 과정은 어땠나요?"
- "AI 기술에 대한 내 관점의 변화는?"
- "가장 자주 다룬 주제는 무엇인가요?"

⚠️ 주의:
- 각 파일은 50만 단어 제한 내로 분할됨
- 한글이 포함된 파일이므로 UTF-8 인코딩 필요
- 파일이 생성되지 않으면 temp/ 폴더의 로그 확인
